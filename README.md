# 👈 Zeigestock Everywhere

![Logo: Explodierender Zeigestock](./assets/icon-128.png)

Extension for all Chromium- and Gecko-based Web Browsers that loads the script
[`https://haesemeyer.dev/scripts/zeigestock.js`](https://codeberg.org/haesemeyer/pages/src/branch/master/scripts/zeigestock.js)
on every page.

This is extremely stupid, it will download everything all of the time, requires an internet connection and if I replace the script with malicious code then that's your problem now.
But have you considered how fun having a *Zeigestock* literally ***everywhere*** is!?!?

## Firefox

Firefox requires signing Extensions.
Either sign your copy or disable `xpinstall.signatures.required` in `about:config`
